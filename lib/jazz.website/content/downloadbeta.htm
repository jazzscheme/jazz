Which version?

If you want to stay closely up-to-date on JazzScheme development or plan on contributing, you should choose the 'devel' version of the binaries. It includes the local .git repository needed to pull the latest changes from the public repository or submit patches of contributions. Else, choose the non-'devel' version which is smaller.
