To build / run / test a sample, open it's project in the workbench and double-click it's product icon (the ruler with a pen icon). This will activate the product and make it the current product. You can then use the Product menu to interact with it.

Note that there is no need to build a sample before running it. Jazz support running any module interpreted. The only exception is modules containing FFI code but no sample uses any FFI at the moment.

Game Server

A sample web server.

You can launch the game server using either of the following methods :

  • Select the Game Server product by double-clicking it and then launch the product using the Product / Debug command. You can use the game server console to control the server. Type help for help.
  • Select the Servers window and double click the Game row to start the server inside the IDE.
